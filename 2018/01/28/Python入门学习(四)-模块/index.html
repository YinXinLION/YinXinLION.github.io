Python入门学习(四)

while循环

先来看下Python的while和for循环
while循环可以在结束时执行一个逻辑，看代码

for循环

主要用来遍历/循环 序列、集合、字典

range(0,10)可以生成0-9的序列 0,1,2,3…
range(0,10,2)第三个参数是步长的意思，0,2,4,6,8

包

包相当于是一个文件夹，里头有很多个.py文件。
包下必须有一个特定的文件__init__.py,如果不包含表示只是一个普通文件夹。
__init__.py的某块名字就是包名，并不是包名.__init__

模块1中有__all__，当模块2引入模块1后，只能使用__all__中含有的变量。

__init__用法

当导入某个包时，__init__.py会自动的执行，所以可以用于初始化。

模块

.py文件就是一个模块。
import 模块名即可导入，使用模块下的东西。
包和模块不会重复被导入。不然效率太低。

不要循环导入,会报错。

类

.py文件里头可以有很多个类。
