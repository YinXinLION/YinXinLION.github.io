Python基础

Python入门学习(二)-组

有序列表

有序列表:可以通过index来访问特定元素的列表。
元组及列表的表示

列表和元组的区别在于：元组是不可变的，列表是可变的，我们可以向列表中添加新的元素，元组却不能。

不可变的：在Python中，字符串、元组都是不可变的。

列表之间可以相加但是不能相乘,相减

有序列表可以使用["1","2","3"][0:2]，但是无序列表就不行了(字典,set)

有趣的发现

額外用法

列表中是否包含1
1 in [1,2,3,4] # True
10 in [1,2,3] # False

查看列表长度
len("hello") # 5

列表最大值/最小值
max([1,2,3,4]) # 4
min([1,2,3,4]) # 1

查看字符的ASCII码
ord('d') # 100

set集合

set有去重复功能，无法下标读取。

集合差、并、交

创建空的set
type(set())

dict字典

字典,在列表中是以键值对的形式存在的{key1:value1,key2:value2},Java中经典的HashMap就是字典。

字典的key只能是数字或者字符串。
字典的value可以使列表形式。
{"a":[1,2,3],"b":1}
