Python入门学习(一)

Python基础数据类型

数字

整数:int
浮点数:float(双精度) 在其他语言中double才是双精度

type函数是用来查看数据类型的

二进制的表示

以0b开头的表示为二进制
以0o开头的表示为八进制
以0x开头的表示为十六进制

bin(num)函数是将数字转换为二进制数
int(num)函数是将数字转换为十进制数
hex(num)函数是将数字转换为十六进制数
oct(num)函数是将数字转换为八进制数

布尔类型

True表示正确.False表示错误.

True即为正数，False即为0

复数

数字+j 表示为复数
36j

字符串

单引号和双引号都能表示字符串。
三个单引号和三个双引号效果是一样的，都能换行输入，字符串结果中带有\n转义符

转义字符

\n 换行
\t 横向制表符

原始字符串

当我们要打印一个windows下的路径时是这样做的

前面带上r后，后面的字符串不是普通字符串而是原始字符串

字符串的操作

字符串是由多个字符组成的, 根据index下标，可以获取到当前的字符.
